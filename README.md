# 🚗 Moghera Auto — Car Maintenance Website

A **modern, animated, and responsive website** designed for a professional car service center.
Built with a focus on **clean UI, smooth animations, and real-world usability**.

## ✨ Key Features

* 🎨 Modern UI Design (Glassmorphism + Gradients)
* ⚡ Smooth Animations & Scroll Effects
* 📱 Fully Responsive (Mobile Friendly)
* 🔧 Car Maintenance Services Showcase
* 🧰 Spare Parts Availability Section
* 💻 Fault Detection System (Diagnostics)
* 📋 Detailed Maintenance Report UI
* 📞 Direct WhatsApp & Call Integration

---

## 🧠 Project Overview

This website was created for **Moghera Auto**, a professional automotive service center that provides:

* Full maintenance for all types of cars
* Advanced fault diagnostics using modern tools
* Availability of all spare parts
* Repair duration from **3 to 10 days** depending on the issue
* A **complete report** after finishing the repair
